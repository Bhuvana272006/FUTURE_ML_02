# 🎫 IT Support Ticket Classification App

## 📌 Project Overview

This project is a **Machine Learning-based web application** that classifies IT support tickets into different categories such as **Hardware, Access, Miscellaneous**, etc.

The model is trained using a ticket dataset and deployed using **Streamlit** for real-time predictions.

---

## 🚀 Features

* Classifies support tickets automatically
* Simple and interactive web interface
* Real-time prediction
* Trained using Machine Learning algorithms
* Displays predicted category instantly

---

## 🧠 Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit

---

## 📂 Dataset

* IT Support Ticket Dataset
* Contains ticket descriptions and their corresponding categories

---

## ⚙️ Model Details

* Algorithm: Logistic Regression
* Feature Extraction: CountVectorizer
* Accuracy: **84.57%**
